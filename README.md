# 🍔 Food Catalog App (Flutter)

Aplikasi mobile berbasis Flutter yang menampilkan katalog makanan dengan UI modern ala GoFood, dilengkapi sistem keranjang (cart), animasi interaktif, dan pengalaman pengguna yang smooth.

## ✨ Fitur Utama

- 🍽️ List menu makanan
- 📄 Detail menu (UI modern)
- 🛒 Cart system (keranjang)
- ➕➖ Quantity control
- 📝 Catatan pesanan
- 💬 Floating cart (popup)
- 📳 Haptic feedback
- 🚀 Animasi "add to cart"
- 💰 Total harga otomatis
- 🎯 UI terinspirasi GoFood / GrabFood
